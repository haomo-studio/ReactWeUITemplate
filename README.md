## 概述

本模板采用的主要组件包括:

* [React](http://reactjs.cn/react/docs/getting-started-zh-CN.html)
* [WeUI](https://github.com/weui/weui)
* [React-WeUI](https://github.com/weui/react-weui/)
* []

### 安装部署

	yarn install

运行:

	npm start

打开浏览器, 访问 http://localhost:3000
